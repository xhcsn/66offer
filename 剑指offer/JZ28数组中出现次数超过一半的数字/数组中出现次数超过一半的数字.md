#### 数组中出现次数超过一半的数字

数组中有一个数字出现的次数超过数组长度的一半，请找出这个数字。例如输入一个长度为9的数组[1,2,3,2,2,2,5,4,2]。由于数字2在数组中出现了5次，超过数组长度的一半，因此输出2。你可以假设数组是非空的，并且给定的数组总是存在多数元素。

```java
public class Solution {
    public int MoreThanHalfNum_Solution(int [] array) {
        int count = 0;
        int x = 0;
        for(int num : array){
            if(count == 0){
                x = num;
            }
            if(num == x){
                count++;
            }else{
                count--;
            }
        }
        return x;
    }
}
```
